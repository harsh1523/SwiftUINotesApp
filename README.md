A secure Notes app built using SwiftUI and MVVM.

## Features
- SwiftUI + MVVM architecture
- Offline persistence (JSON)
- App-level lock using Face ID / PIN
- Per-note lock & unlock
- Minimal Apple-style UI

## Tech Stack
- Swift
- SwiftUI
- MVVM
- LocalAuthentication
- FileManager
